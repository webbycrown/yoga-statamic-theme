# Yoga - Statamic Starter Kit

At Yoga Statamic, we offer a serene and professional website for yoga studios and wellness practitioners, built using Statamic CMS. Designed with tranquility and ease of use in mind, the Yoga starter kit makes it simple to showcase classes, retreats, podcasts, and more through a modern layout and intuitive content management.

## Pages of Yoga

The Yoga starter kit includes a comprehensive set of pages to cover all wellness studio needs:

- **Home Pages**: 4 Variants (Home, Home-Two, Home-Three, Home-Four)
- **About Us Page**
- **Classes**:
  - Classes Listing (2 variants)
  - Class Detail
  - Class Category
  - Book Class
- **Schedule Page**
- **Events**:
  - Events Listing (2 variants)
  - Event Detail
- **Retreats**:
  - Retreats Listing
  - Retreat Detail
- **Blog**:
  - Blog Listing (3 variants)
  - Blog Detail
- **Podcast**:
  - Podcast Listing
  - Podcast Episode Detail
- **Team**:
  - Team Listing
  - Team Member Detail
- **Other Pages**:
  - FAQ Page
  - Testimonial Page
  - Membership Page
  - Privacy Policy
  - Terms & Condition
  - Contact Us
  - 404 Page

## Collections

Organize your content with built-in collections:

- **Pages**: Manage your site structure and hierarchical content.
- **Blogs**: Share news, tips, and wellness articles.
- **Classes**: Define your yoga sessions, styles, and difficulty levels.
- **Events**: Organize workshops and special sessions.
- **Retreats**: Plan and showcase upcoming wellness escapes.
- **Podcast**: Host and manage your audio content.
- **Podcast Episode**: Individual episodes for your podcast series.
- **Teams**: Highlight your instructors and staff.
- **Testimonials**: Showcase student success stories and feedback.
- **FAQs**: Handle common student questions.
- **Comments**: Manage user feedback on your blog posts.

## Features of Yoga

- **Rich Page Builder**: Create unique layouts using 46 flexible sets (hero sections, content blocks, galleries, etc.).
- **Theme-Ready Design**: Modern, premium aesthetics with Tailwind CSS support.
- **Global Settings**: Easily manage site-wide settings, headers, and footers through the Control Panel.
- **Responsive Layout**: Optimized for all devices from mobile to desktop.
- **Statamic 5 Ready**: Built for the latest version of Statamic for maximum performance and security.

## Installation

Follow the [Starter Kit installation instructions](https://statamic.dev/starter-kits/installing-a-starter-kit) to get started with Yoga.
Make sure you're running **Statamic 5.x** for compatibility.

### Installing into an existing site

```bash
php please starter-kit:install webbycrown/yoga-statamic-theme
```

### Installing via the Statamic CLI Tool

If you have the [Statamic CLI Tool](https://github.com/statamic/cli) installed, create a new Statamic installation with Yoga in one command:

```bash
statamic new my-site webbycrown/yoga-statamic-theme
```
